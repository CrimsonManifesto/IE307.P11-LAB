## Prerequisites
- Node.js version 14.x or higher is preferred.
- npm
- Expo CLI
- Android Studio (for Android builds)

# Installation Guide
## Clone the Project
    git clone https://github.com/your-username/your-repository.git
    cd your-repository

## Install Dependencies
    npm install

## Run the Project

### On android
     npm run android

### On iOS:
     npm run ios
